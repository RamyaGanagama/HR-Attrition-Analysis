# HR Attrition Analysis

## Project Overview
End-to-end HR attrition analysis on IBM HR Employee Dataset 
to identify key drivers of employee attrition using 
Excel, SQL, and Power BI.

## Tools Used
- Microsoft Excel — Data Quality Check
- SQL (SQLite) — Data Analysis & Business Insights
- Power BI — Dashboard (In Progress)

## Dataset
IBM HR Analytics Employee Attrition Dataset
- 1,470 employee records
- 35 columns including Age, Department, Salary, 
  Attrition, Job Role

## Key Findings
- Overall Attrition Rate: **16.12%** (237 of 1,470 employees)
- Highest Attrition Department: **Sales at 20.63%**
- Highest Attrition Age Group: **18-25 at 35.77%**
- Highest Attrition Salary Band: **Low earners at 28.61%**
- Highest Attrition Job Role: **Sales Rep at 39.76%**

## Files in Repository
- WA_Fn-UseC_-HR-Employee-Attrition.csv — Raw dataset
- HR_Attrition_Queries.sql — SQL analysis queries
- HR_Dashboard.png — Power BI dashboard (coming soon)

## SQL Queries Performed
1. Overall attrition count and rate
2. Attrition by department
3. Attrition by age group
4. Attrition by salary band
5. Top 5 job roles by attrition rate

## Business Recommendations
1. Focus retention efforts on Sales department (20.63% attrition)
2. Create career development programmes for 18-25 age group
3. Review compensation for low-salary band employees
4. Investigate Sales Representative role satisfaction (39.76% attrition)
